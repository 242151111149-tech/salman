Pengantar Persamaan Kuadrat

Persamaan kuadrat adalah persamaan polinomial derajat dua yang umumnya ditulis dalam bentuk:

ax² + bx + c = 0

di mana a, b, dan c adalah bilangan nyata dan a ≠ 0. Tujuan utama mempelajari persamaan kuadrat adalah mencari nilai x yang memenuhi persamaan tersebut (akar/solusi).

Rumus & Konsep Penting

1. Rumus Kuadrat (Rumus ABC)

Untuk ax² + bx + c = 0, solusi x diberikan oleh:

x = (-b ± √(b² - 4ac)) / (2a)

Diskriminan D = b² - 4ac menentukan banyaknya solusi:

  • D > 0 → dua solusi real berbeda
  • D = 0 → satu solusi real kembar
  • D < 0 → dua solusi kompleks

2. Bentuk Faktorisasi

Jika akar persamaan diketahui (r1, r2), maka:

a(x - r1)(x - r2) = 0

Contoh Soal Diselesaikan

Contoh 1 — Dua akar real

Selidiki akar dari: 2x² - 3x - 5 = 0

Langkah singkat:

  1. Hitung diskriminan D = $D = (-3)^2 - 4·2·(-5) = 9 + 40 = 49$
  2. Karena D = 49 > 0, ada dua akar real.
  3. Akar: x = (3 ± √49)/(4) → x₁ = (3+7)/4 = 10/4 = 2.5; x₂ = (3-7)/4 = -1

Contoh 2 — Akar Kembar

Persamaan: x² - 6x + 9 = 0

  1. D = 36 - 36 = 0 → akar kembar
  2. x = 6 / 2 = 3 → x = 3 (dua kali)

Contoh 3 — Akar Kompleks

Persamaan: x² + 4x + 13 = 0

  1. D = 16 - 52 = -36 < 0 → akar kompleks
  2. Akar: x = (-4 ± √(-36)) / 2 → x = -2 ± 3i

Latihan Interaktif

Masukkan koefisien a, b, dan c untuk melihat akar persamaan.

Hasil akan tampil di sini.

Soal Latihan (Untuk dikerjakan tanpa kalkulator)

  1. Selesaikan: x² - 5x + 6 = 0
  2. Selesaikan: 3x² + x - 4 = 0
  3. Tentukan jenis akar: 4x² + 4x + 1 = 0
